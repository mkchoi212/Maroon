# How to install
```
pod install
```
If you don't have cocoapods installed on your machine, run this command in order to install the gem
```
sudo gem install cocoapods
```
# Requirements
- iOS 8.0 or higher
- ARC

# Structure
The app is divided into four parts
- Home
- Maps
- Food
- Settings
